# 项目名称：6A组 白盒对抗样本攻击与防御实验
## 一、项目简介
本项目基于CIFAR‑10十分类图像数据集与适配32×32图像的轻量化ResNet18卷积网络，完整复现FGSM、PGD、DeepFool三类经典白盒对抗样本攻击，并实现对抗训练、JPEG图像预处理两种防御方案。实验定量探究扰动上限ε、PGD迭代步数对攻击成功率ASR的影响，开展攻击‑防御军备竞赛对照测试，通过折线图、柱状图、样本对比图、t‑SNE特征分布图完成可视化分析，量化对比不同攻防算法的性能差异。实验验证迭代PGD攻击欺骗能力最强，对抗训练可显著提升模型鲁棒性，完整验证白盒对抗样本生成逻辑与两类防御方案的防护效果。

## 六、实验结果
干净 ResNet18 模型在 CIFAR‑10 测试集基准准确率为 83.86%。在扰动 ε=0.0314 条件下，FGSM 攻击成功率达到 90.99%；迭代 10 步的 PGD 攻击成功率可达 100%。DeepFool 采用自适应最小扰动，攻击成功率为 80.10%。经过对抗训练之后的鲁棒模型，面对 PGD 攻击时攻击成功率下降至 54.50%；采用 JPEG 压缩质量为75进行输入防御时，PGD攻击成功率为97.15%，但干净样本准确率下降至79.38%。

核心实验结论：
同等扰动限制下迭代PGD攻击欺骗效果显著优于单步FGSM、最小扰动DeepFool；
扰动ε、PGD迭代步数存在攻击效果饱和阈值，超过阈值后提升参数无明显收益；
对抗训练是可靠长效防御手段，JPEG压缩可轻量化降噪但会损失原始图像识别精度；
微小人眼不可见像素扰动即可偏移模型深层特征，造成完全误分类，模型天然存在对抗脆弱性。
